# BTree, BPlusTree
---
Before start, git clone and compile what data structure you want to see.

### Get start with B-Tree
```
gcc -o BTree BTree.c
./BTree
```

### Get start with B+Tree
```
gcc -o BPlusTree BPlusTree.c
./BPlusTree
```
You can see the input console program.
Check the operations and enjoy yourself!
